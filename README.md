# 🚀 Exploring the Linux Kernel  

This repository is my **playground** for learning, breaking, and experimenting with the Linux kernel.  
It’s not a polished project — it’s more like a **diary of experiments** where I try out new ideas and share what I learn along the way.  

---

## 🌱 What this repo is about  

- A space to **document my learning journey**  
- Small experiments that help me understand how the Linux kernel works  
- Notes, test drivers, and random ideas that I try while exploring  
 
---

## 🎯 Why I’m doing this  

I believe the best way to learn systems programming is by **experimenting hands-on**.  
Every small code snippet or test here represents something I tried, failed at, fixed, or finally understood.  

---

## 📚 Who is this for?  

- Anyone curious about Linux internals  
- Beginners who want to see small kernel code examples  
- Developers who enjoy exploring "how things really work under the hood"
